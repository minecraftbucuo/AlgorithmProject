
# **999题解**
##### 我们先来看一个低级点的问题  

读者肯定知道一个数能被3整除的条件,是去看这个数的各个数位的累加和是否是3的倍数,读者可以简单自行证明一下

这里降低了难度只是询问了是否是9的倍数,是为了让大家即使没见过也能猜出答案,让大家敢于猜想,再去证明其数学上或者逻辑上的正确性,严谨性

##### **类似的,我们猜想一个数能被9整除的条件是,这个数的各数位的和能被9整除**  

证明如下:  
假设 $x =a_n\cdots a_2a_1a_0$ 其中$a_i$表示这个数的第$i$个数位的值
那么$x =\Sigma^{i=n}_{i=0} a_n*10^n$  
我们要去我们去验证这个式子对9取余的结果  

事实上如果有见过要求一些因为答案过大而要求你对答案对一个数$p$取模的题时,你会没有多想的去用上这么一个东西
##### **同余原理**   
1. $(a+b) \%p= (a\%+b\%p)\%p$
2. $(a*b) \%p= ((a\%p)*(b\%p))\%p$  

这两个等式能成为我们计算中途就自信取余的底气,这两个等式读者自证不难,可自行尝试证明

故$(\Sigma^{i=n}_{i=0} a_n*10^n=(\Sigma^{i=n}_{i=0} (a\%9)*(10^n\%9))\%9=(\Sigma^{i=n}_{i=0} a_n)\%p$ 

计算验证即可

##### 参考代码
~~~c++
#include<iostream>
using namespace std;
int main()
{
    int T;
    cin>>T;
    while (T--)
    {
        string str;
        cin>>str;
        int r=0;
        for (int i=0;i<str.size();i++)r=(r+(str[i]-'0'))%9;
        if (r)cout<<0<<endl;
        else cout<<1<<endl;
    }
    return 0;
}
