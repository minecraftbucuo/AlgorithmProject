
冒险

一位叫做鸥的女孩非常喜欢冒险,现在鸥带着她的行李箱来到了洞穴冒险,可惜的是,
鸥的箱子不幸掉进了洞穴中的小溪了,鸥辛苦打捞上她的箱子后发现箱子的密码锁坏掉了,只能以特定方式调大锁上面的初始数n

具体方式如下
你可以执行以下操作任意次
操作1 将锁上的数加上3
操作2 将锁上的数加上2;

只要锁上的数字等于鸥设置密码m,那么箱子就能打开
我们的鸥会告诉你她的密码是什么,而鸥希望尽快帮助她打开箱子,
所以鸥会问你到达鸥设置的密码的最小的操作次数是多少,
如果永远无法打开箱子,鸥会伤心地哭出来,请输出 "QAQ"   (不包含引号)

多组输入数据

input
第一行输入一个正整数T,表示有T组输入数据
接下来每行两个以空格隔开的正整数n,m,表示锁的初始数字和鸥设置的密码
保证 T<=100  , 1<=n<=1e9 ,1<= m<=1e9
output
每组输入数据的答案占一行,表示最小的操作次数,或者是"QAQ"表示无打开箱子



样例1

input 
3
0 3
0 5
5 0
output
1
2
QAQ
