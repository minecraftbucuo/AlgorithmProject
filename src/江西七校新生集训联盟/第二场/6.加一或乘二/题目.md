题目描述：
我们回顾一下，数组 $a$ 的一个递增子序列是指可以通过删除其中一些元素（不改变剩余元素的顺序）得到的子序列，并且剩下的元素严格递增（即 $a_{b_1} < a_{b_2} < ⋯ <a_{b_k}$ 且 $b_1 < b_2 < ⋯ < b_k$ ）。注意，空子序列也被认为是递增的。
现在，给定一个长度为 $n$ 的数组，求这个数组一共有多少个递增子序列。为了让题目更简单，给定的数组将被限制：

1. 数组的第一个元素一定是 $0$ ；
2. 对于第 $i \space (2 \le i \le n)$ 个元素，$a[i] = 0$ 或 $a[i] = 1 + \max_{1 \le j \le i - 1} a_j$ 。 

由于答案可能会很大，请输出其对 $10^9 + 7$ 取模后的结果。

输入描述：
第一行一个整数 $n \space (1 \le n \le 10^5)$ 。
第二行 $n$ 个整数 $a_1, a_2, \dots, a_n$ ，保证数组满足题目所给的限制。

输出描述：
输出一个整数，表示计算得到的答案。
