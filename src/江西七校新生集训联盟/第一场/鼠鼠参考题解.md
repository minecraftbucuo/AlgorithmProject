
# 贪婪的鼠鼠题解
#### 突破点
需要注意到小橘猫的要回食物的规则为
鼠鼠取得的食物相同返回的是烧鸡
取得的食物不同返回的是哦润吉
对应着异或运算的定义

而异或运算可以视为一种不进位的加法 
这意味着异或运算是满足 结合律的 
即 a ^ b ^ c = a ^ ( b ^ c )
这也导致鼠鼠取出食物的顺序不会影响最后结果的

答案即 n个1,m个0异或的结果

#### 参考代码

~~~c++
#include<iostream>
using namespace std;

int main()
{
    int T;
    cin>>T;
    while (T--)
    {
        int n,m;cin>>n>>m;
        if (n&1)cout<<"1.00"<<endl;
        else cout<<"0.00"<<endl;
    }
    return 0;
}
