## 题目描述

Monocarp 正在玩另一款电脑游戏。这一次，他的角色正在击杀一些怪物。有 $n$ 个怪物，编号从 $1$ 到 $n$，第 $i$ 个怪物初始有 $a_i$ 点生命值。

Monocarp 的角色拥有一种技能，可以对当前生命值最高的怪物造成 $k$ 点伤害。如果有多个怪物生命值相同且最高，则选择编号较小的那个。如果某个怪物的生命值在受到伤害后小于等于 $0$，那么它就会死亡。

Monocarp 会不断使用这个技能，直到所有怪物都死亡。你的任务是确定每个怪物死亡的顺序。

## 输入格式

第一行包含一个整数 $t \space (1 \le t \le 10^4)$ ，表示测试用例的数量。

每个测试用例的第一行包含两个整数 $n$ 和 $k$ $(1 \le n \le 3 \times 10^5; 1 \le k \le 10^9)$ ，分别表示怪物的数量和技能造成的伤害值。

第二行包含 $n$ 个整数 $a_1,a_2,…a_n \space (1 \le a_i \le 10^9)$，表示每个怪物的初始生命值。

所有测试用例中 $n$ 的总和不超过 $3 \times 10^5$。

## 输出格式

对于每个测试用例，输出 $n$ 个整数，表示怪物按照死亡顺序的编号。

## 输入输出样例

**输入**

```
3
3 2
1 2 3
2 3
1 1
4 3
2 8 3 5
```

**输出**

```
2 1 3 
1 2 
3 1 2 4
```
