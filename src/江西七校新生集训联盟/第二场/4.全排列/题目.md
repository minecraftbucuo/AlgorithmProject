### **题目描述**：

给定 $n$ ，输出所有长度为 $n$ 的全排列，按字典序从大到小输出。

已知如下是按字典序从小到大输出的代码：

```C++
#include <iostream>
using namespace std;
const int N = 10;

int n, a[N], vis[N];

void dfs(int deep) {
    if (deep == n) {
        for (int i = 0; i < n; i++) cout << a[i] << ' ';
        cout << endl;
        return;
    }
    for (int i = 1; i <= n; i++) {
        if (!vis[i]) {
            vis[i] = 1;
            a[deep] = i;
            dfs(deep + 1);
            vis[i] = 0;
        }
    }
}

int main() {
    cin >> n;
    dfs(0);
    return 0;
}
```

#### **输入格式**

一行一个整数 $n \space (1 \le n \le 8)$

#### **输出格式**

每种全排列占一行，共 $n!$ 行；每行的数用空格分隔。

#### **样例：**

- **输入**：

  ```
  3
  ```
  
- **预期输出**：

  ```
  3 2 1
  3 1 2
  2 3 1
  2 1 3
  1 3 2
  1 2 3
  ```
